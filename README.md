# carteav - cinemas

## Run Via Docker Compose

> ### Prerequisite
> Install [docker desktop](https://docs.docker.com/desktop/).

  - To run the application:

    ```
    docker-compose up --build -d
    ```
    
    The application opens at http://localhost:3001

  - To stop the application:

    ```
    docker-compose down
    ```
